## 项目现状概览

### 当前技术栈
- **前端框架**: Next.js 14 + React 18 + TypeScript
- **状态管理**: Jotai + IndexedDB (Dexie.js)
- **UI 组件**: Radix UI + Tailwind CSS + Framer Motion
- **Markdown**: react-markdown + Shiki + KaTeX + Mermaid
- **国际化**: next-intl (支持中文/英文/日文)

### 已有核心功能
✅ 实时聊天界面（流式/非流式）
✅ 多角色消息管理（system/user/assistant）
✅ 消息拖拽排序和内联编辑
✅ 图片/文件上传和预览
✅ Markdown 完整渲染（GFM + 数学公式 + Mermaid 图表）
✅ 词元概率显示（OpenAI 模型）
✅ 80+ 模型选择和参数配置
✅ 新手/专家双模式
✅ 深色/浅色主题切换
✅ IndexedDB 本地持久化
✅ 多语言国际化支持
✅ 导出为 Markdown 文件

**优先级**: ⭐⭐⭐⭐⭐
**难度**: 中等

**功能描述**:
